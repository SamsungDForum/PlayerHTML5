Tizen Video App

Video player based on HTML5 video tag

Available buttons/actions:

1 - change source

Info

MP4 files can be tested in Tizen Emulator. Other formats need TV.

The application downloads video content from http://media.w3.org - it will not work if this domain is not accessible
You can change the source for video content by modifying contents of sources array in main.js file

Operation

Use remote to operate on video player.
Available keys: PLAY, PAUSE, FW, RW, STOP, RETURN
